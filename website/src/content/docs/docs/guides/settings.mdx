---
title: Settings & Theming
description: Themes, layout, notifications, keybindings, language, and integrations — all live, all persisted.
---

Open with the **Menu** button (top right) or `Ctrl+Space ,`. Every change
applies live and persists to `~/.bohay/config.json`.

| Tab | What's there |
|---|---|
| **Theme** | 10 built-in palettes (dark and light) |
| **Layout** | pane gaps, titles, sidebar width, resume-into-workspace behavior |
| **Notifications** | desktop alerts on *blocked* / *done* (always silent — no terminal bell) |
| **Keys** | every command's binding — click one, press the new key |
| **Modules** | installed extensions: enable/disable, actions, logs |
| **Integrations** | install/uninstall the per-agent session hook |
| **Language** | the UI in 8 languages (en · es · pt · fr · de · id · zh · ja) |

Keybinding notes: arrows and `hjkl` both work for focus; `Tab`/`Shift+Tab`
cycle tabs; every prefix command is remappable, and `Ctrl+Space ?` always
shows the current map.
