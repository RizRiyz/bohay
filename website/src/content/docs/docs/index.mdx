**bohay** is a terminal workspace for AI coding agents: split panes, a live
sidebar of every agent's state, sessions that survive closing the terminal, and
tools to run many agents on one project without conflicts.

## Install

<Tabs syncKey="install">
  <TabItem label="macOS / Linux">
    ```sh
    curl -fsSL https://bohay.dev/install.sh | sh
    ```
    Prebuilt binaries for Apple silicon and Linux (x86_64 / arm64).
  </TabItem>
  <TabItem label="Homebrew">
    ```sh
    brew install RizRiyz/bohay/bohay
    ```
  </TabItem>
  <TabItem label="Cargo">
    ```sh
    cargo install bohay
    ```
    Any platform with Rust ≥ 1.82, including Intel macs.
  </TabItem>
  <TabItem label="Windows">
    ```powershell
    irm https://bohay.dev/install.ps1 | iex
    ```
    Use bohay inside **Windows Terminal**.
  </TabItem>
</Tabs>

## Run it

```sh
bohay          # launch or reattach to your session
bohay doctor   # check your setup: git, gh, ssh, and what each unlocks
```

That's genuinely it. `bohay` starts a background server that owns your panes and
attaches a lightweight client. Close the terminal any time and everything keeps
running. Start your agents in panes like you normally would, and bohay recognizes
them automatically and shows their live state in the sidebar.

:::caution[macOS: free the prefix key]
macOS reserves `Ctrl+Space` for input-source switching, which blocks bohay's
prefix. Untick *Select the previous input source* under **System Settings →
Keyboard → Keyboard Shortcuts… → Input Sources**. Everything is also fully
mouse-driven, so nothing else is blocked in the meantime.
:::
